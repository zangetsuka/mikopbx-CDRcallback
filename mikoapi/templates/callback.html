Конфигурация автоматического обратного звонка
Задачи обратного звонка
Состояние модуля
Включение автоматического обзвона и создания задач из пропущенных вызовов.
Модуль включён Запускает фоновый обзвон по расписанию
Авто-создание задач из CDR Новые пропущенные автоматически попадают в очередь обзвона
Защита от дублей и устаревания
Не звонить повторно по одному номеру и не «доживать» задачам до следующего дня.
0 — выключено. В пределах окна повторная задача по номеру не создаётся.
0 — выключено. Задачи старше этого срока истекают вместо звонка.
Куда направлять звонок
Выбор адресата обратного звонка: очередь операторов или конкретный внутренний номер.
Очередь — звонок идёт в очередь; Внутренний — напрямую оператору.
Канал и идентификация
Технические параметры исходящего плеча и номер, который увидит клиент.
{phone} подставляется автоматически. Напр. `PJSIP/{phone}` или `Local/{phone}@outgoing`.
Номер, который увидит клиент и который попадёт в CDR как «От». Обычно ваш городской DID.
Повторные попытки
Сколько раз и с какими паузами перезванивать, если клиент не ответил.
Используется, если backoff-расписание пустое.
Нарастающие паузы, через запятую. Например `5,15,30`.
Тайминг звонка
Задержки запуска и таймауты дозвона.
Голосовое уведомление
Аудио-сообщение, которое проигрывается клиенту при обратном звонке.
Файл проигрывается перед соединением с оператором.
Рабочее время
Обзвон выполняется только в разрешённые окна. Можно задать несколько интервалов, например отдельно до и после обеда.
Звонить только в рабочее время Вне активных интервалов задачи остаются в очереди до следующего разрешённого окна.
Интервалы обзвона Пример: `09:00–13:00` и `14:00–18:00` по будням.
Интервалы не добавлены. Добавь хотя бы один интервал или отключи ограничение по рабочему времени.
Защита от перегрузки
Ограничение интенсивности обзвона для защиты АТС от шторма вызовов.
0 — без лимита
0 — без лимита
Проверка занятости оператора
Не инициировать звонок, если оператор сейчас разговаривает или недоступен.
Проверять занятость оператора Откладывает задачу без расхода попытки, если оператор занят.
Пусто — берётся номер оператора из маршрутизации.
Каналы уведомлений
Внешние оповещения о событиях обзвона. Появятся в следующих версиях: Telegram, e-mail, webhook.
Каналы уведомлений пока не настроены
Обслуживание базы данных
